# Heap



## Methods

- [x] `add()`: Add element to the heap.
- [x] `poll()`: Remove root element and return it.
- [x] `peek()`: Get the root element of the heap.
- [ ] `isEmpty()`: Check if heap is empty.


## Complexity

#### Time Complexity

| Methods      | Complexity |  
| :----------: | :--------: |  
| `add()`      | O(n)       |  
| `poll()`     | O(n)       |  
| `peek()`     | O(1)       |
